# blinkit-retail-analytics-dashboard

Blinkit Retail Analytics Dashboard
Project Overview

This project presents a retail analytics dashboard created using Microsoft Power BI Desktop to analyze sales, product demand, pricing patterns, and delivery performance of the Blinkit quick-commerce platform. The dashboard converts raw operational data into interactive visual insights that help understand customer ordering trends and business performance.

Problem Statement

Quick-commerce platforms generate large volumes of order, product, and delivery data. However, extracting meaningful insights from raw data is difficult. This project aims to analyze Blinkit retail data and visualize key metrics such as total orders, revenue, product demand, pricing differences, and delivery performance through an interactive Power BI dashboard.

Dataset

The project uses multiple datasets including:

blinkit_orders.csv

blinkit_order_items.csv

blinkit_products.csv

blinkit_delivery_performance.csv

These datasets contain information related to orders, products, sales value, and delivery performance.

Tools Used

Microsoft Power BI Desktop

Data Visualization

Basic Data Modeling

Dashboard Features

The dashboard includes multiple visual insights such as:

Total Orders, Total Revenue, and Total Products

Sales distribution by product category

Top selling products analysis

MRP vs Actual Price comparison

Delivery status distribution

Order trends over time

Key Insights

Identified the most demanded products and categories.

Analyzed pricing differences between MRP and selling price.

Evaluated delivery performance using delivery status data.

Observed overall sales and order patterns.

Conclusion

The Blinkit Retail Analytics Dashboard demonstrates how business data can be transformed into meaningful insights using Power BI. The dashboard helps understand product demand, pricing strategies, and operational performance, supporting better decision-making in retail and quick-commerce environments.
